# :spiral_notepad: dotfiles - configuration files

## :arrow_right: How use
```zsh
cd ~
git clone git@github.com:CarlosKvasir/dotfiles.git .dotfiles

cd .dotfiles
stow nvim
```
Stow create symlinks using same path of .dotfiles
